🚀 Solar Power Forecasting and Fault Detection

Hybrid Deep Learning Approach using CNN + LSTM + Transformer

Built a highly accurate, end-to-end solar power forecasting pipeline using deep learning to tackle real-world energy optimization. The model integrates weather and inverter data, processes it using a hybrid CNN–LSTM–Transformer architecture, and supports real-time predictions (minute-wise and hourly) with fault detection capabilities.

📌 Key Features:

  • Hybrid architecture combining CNN (spatial features), LSTM (temporal patterns), and Transformer (global attention)
  • Preprocessing of sensor data: GHI, POA, module temp, wind speed, humidity, inverter power
  • Prediction of Active Power (kW), DC Power, and Today’s Generation (kWh)
  • Correlation heatmaps and anomaly detection for real-time fault alerts
  • Performance: MAE: 3.25%, MSE: 0.257%
  • Presented at GCDEM 2024 for its contribution to renewable energy systems

🛠️ Tech Stack: TensorFlow, Keras, Scikit-learn, Python, Matplotlib

🧱 Ultimeet – AI-Powered Meeting Assistant

Redefining Digital Meetings with AI + Voice Biometrics

Ultimeet is an end-to-end intelligent meeting platform that revolutionizes online meetings. It takes recorded audio/video from virtual meetings, transcribes them with high accuracy, separates speakers using biometric voice signatures, and generates detailed outputs like MoMs, action items, key decisions, and engagement analytics—all automatically.

📌 Key Capabilities:

  • Voice Biometric Speaker Identification: Each participant’s voice is uniquely recognized and tagged using advanced biometric fingerprinting, ensuring clarity and accountability.

  • Action Items Extraction: Automatically detects task-related commitments and assigns them within an integrated task manager for tracking and follow-up.
  • Advanced Analytics: Visualizes speaker participation, topic durations, and sentiment insights using data-driven dashboards to optimize meetings.
  • Seamless Integrations: Works smoothly with CRMs, calendars, and communication platforms to embed AI features into everyday workflows.
