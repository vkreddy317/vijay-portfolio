🚀 Solar Power Forecasting and Fault Detection

Hybrid Deep Learning Approach using CNN + LSTM + Transformer

Built a highly accurate, end-to-end solar power forecasting pipeline using deep learning to tackle real-world energy optimization. The model integrates weather and inverter data, processes it using a hybrid CNN–LSTM–Transformer architecture, and supports real-time predictions (minute-wise and hourly) with fault detection capabilities.

📌 Key Features:

  • Hybrid architecture combining CNN (spatial features), LSTM (temporal patterns), and Transformer (global attention)
  • Preprocessing of sensor data: GHI, POA, module temp, wind speed, humidity, inverter power
  • Prediction of Active Power (kW), DC Power, and Today’s Generation (kWh)
  • Correlation heatmaps and anomaly detection for real-time fault alerts
  • Performance: MAE: 3.25%, MSE: 0.257%
  • Presented at GCDEM 2024 for its contribution to renewable energy systems

🛠️ Tech Stack: TensorFlow, Keras, Scikit-learn, Python, Matplotlib
